# Relationship Diagram Tool (English Unofficial Translation and Reconstruction)

A free, browser-based character relationship chart maker. Create cute pastel diagrams for anime, manga, dramas, friend groups, and more, no signup, no installation required.

**Original Japanese Tool:** https://amix-design.com/tl/tool-dia/

---

## Features

- **Image Upload for Characters** - upload photos or use emojis/colored circles, shapeable into circles, squares, or bubbles
- **6 Arrow Styles** - one-way, two-way, heart, cracked, dotted, and wavy lines
- **Relationship Labels & Stamps** - presets like Love, Hate, Rival, Trust, Interested, plus custom labels and decorations
- **Unlimited Nodes** - add as many characters as needed and drag them freely on the canvas
- **PNG / JPEG Export** - one-click image save ready for social media or documents
- **Undo / Redo** - Ctrl+Z / Ctrl+Y support
- **Save / Load Projects** - download as a `.sankaku.json` file and resume editing later (includes uploaded images)
- **Auto Backup** - work is automatically saved to your browser's local storage
- **Sample Diagrams** - load pre-made examples to get started quickly

## Usage

1. Open `index.html` (or the local server URL).
2. Click **"+ Add Node"** to add a character and give them a name.
3. Set the icon, upload an image, pick an emoji, or choose a colored circle.
4. Hover over a node and drag the **+** handle to another node to create a connection.
5. Click on a line to customize the relationship label, arrow style, color, and decorations.
6. Export your diagram as **PNG** or **JPEG** using the toolbar buttons.
